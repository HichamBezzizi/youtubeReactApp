YoutubeAppReact:

- Has a search bar where you can search for any video.
- Displays 1 video and a list of videos.
- The list of videos are clickable.

Tools used:

- React (with hooks)
- HTML/CSS/Javascript
- Material-UI: 
        - Grid 
        - Paper
        - Typography
- Youtube API
- Axios
